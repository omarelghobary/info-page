HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content structure and meaning we don't use it for styling purposes. Use css for that instead!

  • HTML uses Elements to describe content
  • HTML elements typically have an opening tag, content then closing tag
  • You can also have void (empty) elements like imgages
  • You can also configure elements with attributes
  • There is along list of available elements, you will get comfortable with time, no worries

Learn more about all available HTML elements on MDN HTML element reference

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content

  • Styles are assigned via property-value-pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplication, you typically use global styles (via the style element)
  • Alternatively, you can work with external stylesheet files which you link to
  • when working with css, concepts like "inheritance", "specificty" and the "box mode" should be understood

Learn more about all available css properties and values on the MDN CSS property reference
